Project Overview:
This project is a Machine Learning-based SMS Spam Detection System that classifies text messages as either Spam or Legitimate. The model is trained using the SMS Spam Collection Dataset and utilizes TF-IDF Vectorization for text feature extraction and Support Vector Machine for classification. Users can enter messages to check whether the received messages are spam or ham.

Features of the Project:
1.Detects whether an SMS is Spam or Legitimate.
2.Uses TF-IDF to convert text into numerical data.
3.Trains a Support Vector Machine Model.
4.It predicts whether the SMS are Spam or ham.
5.This Project is simple and user friendly.
 
Technologies Used:
1.Python
2.Pandas
3.Scikit-learn 
4.TF-IDF Vectorizer 
5.Support Vector Machine (SVM) 
6.SMS Spam Collection Dataset 

How to Run:
1.Clone the Repository.
2.Install Required Libraries-pip install pandas scikit-learn.
3.Download the spam.csv dataset and place it in the project directory.
4.Run the Program.
5.Enter any SMS message when prompted.

Output:
When a user enters an SMS message, the system predicts whether it is spam or legitimate.
